# Projet de Stage – Application de Gestion des Bobines (FIFO)

## Contexte
Ce projet a été réalisé lors de mon **stage chez SEBN, Tunisie**.  
L’objectif était de développer une solution automatisée pour gérer les bobines dans l’atelier en suivant le principe **FIFO (First In, First Out)** afin d’améliorer la traçabilité et l’efficacité des stocks.

## Objectifs
- Mettre en place une gestion automatisée des bobines selon FIFO  
- Améliorer la traçabilité et l’utilisation des stocks  
- Fournir un guidage LED pour faciliter le retrait et le stockage des bobines  
- Développer une application web et back-end performante

## Méthodologie
- Analyse des besoins et modélisation du processus FIFO  
- Développement front-end en **HTML, CSS, JavaScript**  
- Développement back-end en **Node.js** avec **REST APIs** pour la communication  
- Tests fonctionnels pour valider la traçabilité et l’efficacité

## Outils utilisés
- JavaScript, HTML/CSS  
- Node.js  
- REST APIs  
- GitHub pour le suivi du projet

## Résultats
- Application fonctionnelle permettant de gérer automatiquement les bobines selon FIFO  
- Guidage LED opérationnel pour faciliter la manipulation des bobines  
- Amélioration de la traçabilité et de l’efficacité de l’atelier
